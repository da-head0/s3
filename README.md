# s3
s3같은 object storage 만들기

## object storage란
- 데이터 블록보다는 오브젝트(개별 유닛) 관리 (데이터, 메타데이터, 고유한 식별자)
- API 또는 http/https 를 통해 즉시 액세스 할 수 있음 (데이터 보호)
- 필요한 경우 여러 데이터센터에 복제 가능

## 구현 방법
- 디바이스 수준
- 시스템 수준
- 인터페이스 수준
